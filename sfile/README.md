# Sfile Scraper

API for searching and downloading files from Sfile.mobi.

## Features
- **Search**: Search files on Sfile.
- **Download Info**: Extract download link and metadata.
- **Stream**: Stream file directly via proxy.

## Endpoints

### Search
`GET /sfile/search?q=minecraft`

### Download Info
`GET /sfile/download?url=https://sfile.mobi/...`

### Stream
`GET /sfile/stream?url=https://sfile.mobi/...`
